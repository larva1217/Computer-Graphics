# 🖥️ Computer Graphics

## 📚 교과목 개요

3차원 컴퓨터 그래픽스의 원리에 대해 공부하고, 이에 대해 프로그래밍 실습을 진행합니다.

3차원 객체를 생성하는 모델링 기술부터, 객체에 입체감을 표현하기 위한 채색과 명암을 적용하는 렌더링 기술을 학습합니다.

최종적으로 프로그래밍을 이용하여 컴퓨터 그래픽스 이론들을 직접 구현하고, 3차원 장면을 제작합니다.

본 교과목은 (주)더픽트와 공동 운영하는 교과목으로, 우수 기말 프로젝트 출품자는 (주)더픽트에 견학을 가서 발표를 진행할 예정입니다.

---

## 🎯 교과 목표

본 교과목에서는 3차원 장면이 어떠한 파이프라인을 거쳐 2차원 화면에 출력되는지 그 과정을 학습합니다.

또한 3차원 장면 생성과 출력을 도와주는 그래픽 라이브러리를 이용한 프로그래밍을 공부합니다.

기말 프로젝트로 자신만의 3차원 장면을 제작하고 발표합니다.
